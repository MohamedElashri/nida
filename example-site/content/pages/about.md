# About

Nida is a small static site generator for people who want a predictable publishing workflow without a lot of framework ceremony.

It is designed around a few principles:

- plain Markdown content with TOML front matter
- a tiny command surface with `build` and `serve`
- deterministic output that is easy to reason about
- internal modularity without a plugin marketplace

The example site exists to show what a compact but polished Nida project can look like.
