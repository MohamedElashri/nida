# Colophon

This example site is intentionally simple.

- Content lives in `content/posts` and `content/pages`
- Templates live in `templates`
- Static assets live in `static`
- Output is written to `public`

The goal is not to be flashy. The goal is to give you a clean starting point that still feels like a real publication.
